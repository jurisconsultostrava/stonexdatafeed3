StoneX → Shoptet Feed Tool

Interní nástroj pro stažení StoneX PDF katalogu, spárování přes Product number = CODE, ruční kontrolu cen a generování Shoptet supplier XML.

1. Přístup

Pokud není APP_PASSWORD na Railway nastaveno, pole může zůstat prázdné.

2. Supplier XML

Alternativně backend použije SHOPTET_SUPPLIER_FEED_URL z Railway Variables.

3. StoneX PDF katalog

Správný vstup je PDF endpoint, např. /api/client/catalog/pdf/..., ne běžná HTML search stránka.

4. Cenový model

Prodejní cena = StoneX cena × (1 + marže dle kategorie). Nákupní cena = StoneX cena. Pokud StoneX vrátí cenu v EUR, přepočítá se kurzem níže.
